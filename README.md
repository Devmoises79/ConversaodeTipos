# 🔄 Conversão de Tipos em Python

## 📌 Visão Geral
Este projeto demonstra o comportamento da conversão de tipos em Python, também conhecido como *type casting*. O objetivo é entender como diferentes tipos de dados podem ser convertidos e como isso impacta operações matemáticas e lógicas.

---

## ⚙️ Funcionalidades
- Conversão de string para inteiro
- Conversão de string para float
- Conversão de string para booleano
- Operações matemáticas com tipos convertidos
- Exibição do tipo de dado em tempo de execução

---

## 🧠 Conceitos Abordados
- Tipos de dados em Python
- Conversão explícita de tipos (type casting)
- Funções `int()`, `float()`, `bool()`
- Avaliação de valores booleanos em strings
- Comportamento de valores vazios vs não vazios
- Uso de `type()` para inspeção de tipos
